## 📌 Overview

This project is part of the Generative AI Internship Week 4 Assignment. It demonstrates the practical implementation of AI-powered image generation, voice generation, and voice assistant development using open-source Generative AI tools.

# 📚 Part D — Research Component

## Stable Diffusion vs DALL·E vs Midjourney

| Parameter            | Stable Diffusion | DALL·E    | Midjourney |
| -------------------- | ---------------- | --------- | ---------- |
| Cost                 | Free             | Paid      | Paid       |
| Open Source          | Yes              | No        | No         |
| Image Quality        | Excellent        | Excellent | Excellent  |
| Customization        | Very High        | Medium    | Medium     |
| Hardware Requirement | High             | Low       | Low        |

---

## Research Conclusion

Stable Diffusion provides full customization and local deployment capabilities, making it ideal for developers and researchers. DALL·E offers powerful cloud-based image generation with ease of use, while Midjourney specializes in artistic and visually appealing image creation. Each tool serves different user requirements depending on flexibility, cost, and computing resources.
